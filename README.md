# AI Grant 포트폴리오 리서치 목록

이 저장소는 AI Grant가 투자한 스타트업들의 심층 분석 리포트를 정리합니다. 각 항목은 `research/[회사명]_analysis.md` 문서로 연결됩니다.

- 공식 사이트: https://aigrant.com/
- 포트폴리오: https://aigrant.com/portfolio

## 사용 방법
- 아래 회사명을 클릭하면 해당 회사의 분석 문서로 이동합니다.
- 링크는 실제 파일명과 1:1로 매칭되어 동작하도록 정리되었습니다.
